# bitdrift demos and utilities

This repo collects small apps and scripts used to demonstrate bitdrift features.

## Projects

- [Sankey demos](sankeydemo/) - Android, iOS, and React Native demo apps for user journey visualization.
- [Symbol upload](symupload/) - Upload iOS dSYM files with curl (no CLI required).

## Quick navigation

- Android demo apps: [sankeydemo/android/](sankeydemo/android/)
- iOS demo apps: [sankeydemo/ios/](sankeydemo/ios/)
- React Native demo app: [sankeydemo/react-native/sankey-demo-no-sdk/](sankeydemo/react-native/sankey-demo-no-sdk/)
